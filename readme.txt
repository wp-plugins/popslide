Popslide is better WordPress popup plugin which displays fully customisable bar whereever you want.

== Description ==

Display popslide and increase your newsletter signups conversions!

This plugin comes with built in integration with Mailpoet newsletter plugin however you can style you'r own forms by adding custom CSS via beautiful CSS editor

Available languages:

*   English (author)
*   Polish (author)

Want to use this plugin in your language? Send me your translation and get your website link here!

== Installation ==

Don't download .zip file - is silly when you can search for this plugin in your WordPress!
(unless you need .zip)

1. Go to Plugin -> Add New
1. Search for Popslide
1. Install and enjoy!
